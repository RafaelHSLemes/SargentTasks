# SargentTasks
Task Manager System by RafaelHSLemes 08/02/2024

Sargent Tasks system development project planning

Objectives setting

Starting by defining the SMART (Specific, Measurable, Attainable, Relevant and Time-bound) objectives of the project.

What problem does the project solves?

Disorganization when there are too many tasks to do and the user is undecided on how to start.

What features will it have?

- Registering, editing and deleting tasks;

- Categories and tags to organize tasks;

- Priority setting;

- Filters and searches to easily find tasks;

- E-mail and web push notifications (optional)
